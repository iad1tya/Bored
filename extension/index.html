How to Install

  1. Download: Click "Download Extension (ZIP)". You'll need to create a ZIP file of your extension folder (containing `manifest.json`, `background.js`, `content.js`, and `assets`).
  2. Unzip: Extract the downloaded file to a permanent folder on your computer.
  3. Chrome Extensions: Open chrome://extensions in your browser.
  4. Developer Mode: Toggle on the "Developer mode" switch in the top right corner.
  5. Load Unpacked: Click the "Load unpacked" button that appears.
  6. Select Folder: Choose your unzipped extension folder from the file dialog.
  7. Done! The "Bored" extension will now appear in your list. Test it by playing some media!
